Weather App Project

The Weather App provides accurate, real-time weather updates for any location, offering detailed information about the current weather conditions, forecasts, and temperature. This project integrates the OpenWeather API for reliable and up-to-date weather data. Developed using Python and Flask for the backend, it features a responsive frontend built with Bootstrap to deliver a smooth user experience.

  • Real-time weather updates fetched from the OpenWeather API.
  • Displays current temperature, humidity, wind speed, and weather condition.
  • Forecast for the next 5 days with detailed daily weather data.
  • Search functionality to get weather details for any city worldwide.
  • Responsive design for mobile and desktop, ensuring accessibility on all devices.
